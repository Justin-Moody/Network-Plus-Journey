# Network+ Labs

Hands-on labs using Cisco Packet Tracer while studying for the CompTIA Network+ certification.

## Labs
- 1.1 OSI Model and Networking Devices
- 1.2 Networking Functions
- 1.3 Cloud Basics
- 1.4 IP, Ports, and Protocols

## Tools Used
- Cisco Packet Tracer
- Command Prompt / PowerShell
- Wireshark (optional)

## Goal
- Build practical networking skills
- Prepare for Network+ exam
- Create a portfolio for entry-level IT roles
